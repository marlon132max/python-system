# python-system
# Sistema de Cadastro em Python

Projeto de estudo feito em Python utilizando Tkinter para interface gráfica
e MySQL para armazenamento dos dados.

Funcionalidades
- Cadastro de usuários
- Armazenamento no banco de dados
- Geração automática de ID
- Interface gráfica simples

Tecnologias
- Python
- Tkinter
- MySQL (XAMPP)
- mysql-connector-python

Como executar
1. Instale o Python
2. Instale o XAMPP e ative o MySQL
